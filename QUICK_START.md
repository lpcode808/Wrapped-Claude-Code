## What Do These Numbers Mean?

### Messages
**What you typed** - Each time you sent something to Claude Code
- NOT tool calls
- NOT Claude's responses
- Just YOUR inputs

### Sessions
**Individual conversations** - Each session = one sitting
- Opening Claude Code = new session
- Each session = one `.jsonl` file

### Agents
**Background tasks** - Parallel processing Claude spawned
- Advanced feature
- Most users: 0 agents
- Power users: 1-2 agents per session
