# Contributing to Claude Code Wrapped

Thanks for your interest in improving Claude Code Wrapped!

## How to Contribute

### Found a Bug?
- Check if it's already reported in Issues
- If not, open a new issue with:
  - Your OS and Claude Code version
  - Steps to reproduce
  - Expected vs actual behavior

### Have an Idea?
We'd love to hear about:
- New metrics to track
- Better visualizations
- Export formats (HTML, PDF, etc.)
- Integration ideas
- UI improvements

### Want to Code?
1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a PR with clear description

## Development Setup

```bash
git clone https://github.com/yourusername/claude-code-wrapped
cd claude-code-wrapped
# No dependencies needed! Just Python 3 and bash
python3 count_user_inputs.py  # Test it works
```

## Code Style

- Python: Follow PEP 8
- Bash: Use shellcheck
- Comments: Explain WHY not WHAT
- Keep it simple - users should understand the code

## Testing

Before submitting:
- [ ] Scripts run without errors
- [ ] Works on fresh Claude Code install
- [ ] No hardcoded paths or personal data
- [ ] Clear error messages
- [ ] Documentation updated

## Ideas for Contributors

**Easy:**
- Add more fun facts to Spotify Wrapped style
- Improve error messages
- Add examples to README
- Fix typos/documentation

**Medium:**
- Create HTML export
- Add data visualization charts
- Track additional metrics
- Improve timestamp parsing

**Hard:**
- Web dashboard
- Year-over-year comparison
- Team/organization analytics
- Git integration
- Real-time tracking

Open an issue or discussion! We're here to help.
